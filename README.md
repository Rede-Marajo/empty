# is-empty

## _Define uma expressão, de qualquer tipo de dado, para ser verificada se está vazia._

## Instalação

```sh
npm i is-empty
```

Ou

```sh
yarn add is-empty
```

## Uso

```js
const is-empty = require('is-empty');
let vazioArray = [];
let vazioString = '';

if(vazioArray || vazioString){
    console.log('is-empty')
}

```
